# 0089-Go-指针

## 环境

- Time 2022-08-23
- Go 1.19

## 前言

### 说明

参考：<https://gobyexample-cn.github.io/pointers>

### 目标

使用 Go 语言的指针。

## 示例

```go
package main

import "fmt"

func zeroval(ival int) {
    ival = 0
}

func zeroptr(iptr *int) {
    *iptr = 0
}

func main() {
    i := 1
    fmt.Println("initial:", i)

    // 值传递，不会修改原变量的值
    zeroval(i)
    fmt.Println("zeroval:", i)

    // 地址传递，可以修改原变量
    zeroptr(&i)
    fmt.Println("zeroptr:", i)

    fmt.Println("pointer:", &i)
}
```

## 总结

使用 Go 语言的指针。
