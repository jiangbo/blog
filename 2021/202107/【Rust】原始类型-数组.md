# 【Rust】原始类型-数组

## 环境

- Rust 1.54.0
- VSCode 1.59.1

## 概念

数组（array）是一组拥有相同类型 T 的对象的集合，在内存中是连续存储的。  
数组使用中括号 `[]` 来创建，且它们的大小在编译时会被确定。  
数组的类型标记为 `[T; size]`，T 为元素的类型，size 表示数组的大小。

## 示例

### 类型申明

```rust
fn main() {
    let a = [1, 2, 3, 4];
    let b: [i32; 4] = [1, 2, 3, 4];

    println!("{:?}", a);
    println!("{:?}", b);
}
```

### 初始化

```rust
fn main() {
    let a: [i32; 4] = [44; 4];
    println!("{:?}", a);
}
```

### 取值

> 下标从 0 开始

```rust
fn main() {
    let a = [1, 2, 3, 4];
    println!("{:?}", a[3]);
}
```

### 获取长度

```rust
fn main() {
    let a = [1, 2, 3, 4];
    println!("{}", a.len());
}
```

### 越界

如果数组越界，将会引发错误。

```rust
fn main() {
    let a = [1, 2, 3, 4];
    println!("{}", a[4]); // 要报错
}
```

介绍了 Rust 中数组的概念和一些使用方式。
