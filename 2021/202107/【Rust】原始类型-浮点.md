# 【Rust】原始类型-浮点

## 环境

- Rust 1.54.0
- VSCode 1.59.1

## 概念

浮点类型有 `f32` 和 `f64` 两种，四则运算也适用于浮点类型。

> 默认类型是 `f64`，因为在现代 CPU 中，它与 `f32` 速度几乎一样，不过精度更高。

## 示例

### f32

```rust
fn main() {
    let x: f32 = 6.4;
    let y = 4.6f32;
    println!("x = {}, y = {}", x, y)
}
```

### f64

```rust
fn main() {
    let x = 4.6;
    let y: f64 = 6.4;
    let z = 4.6f64;
    println!("x = {}, y = {}, z = {}", x, y, z)
}
```

介绍了 Rust 中的浮点类型的一些概念和操作。
