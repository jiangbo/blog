# 1145-怪物战争-地图背景颜色

## 目标

解析 tiled 地图中定义的背景颜色，然后渲染时使用这个颜色。

## 环境

- Time 2026-02-01
- Zig 0.15.1

## 参考

1. <https://www.bilibili.com/video/BV1jf9XYQEhW>

## 想法

使用了工具，提前将 tiled 地图解析为程序需要的 ZON 格式，然后直接读取。

## scene.zig

新增了一个场景 scene 模块。

```zig
const std = @import("std");
const zhu = @import("zhu");

const tiled = zhu.extend.tiled;

const level: tiled.Map = @import("zon/level1.zon");

pub fn init() void {
    tiled.backgroundColor = level.backgroundColor;
}

pub fn deinit() void {}

pub fn update(delta: f32) void {
    _ = delta;
}

pub fn draw() void {}
```
