## context.zig

将 `Unit` 放到 `context.zig`，让 HUD 和部署逻辑使用同一份单位数据。

```zig
pub const Unit = struct {
    face: u32,
    class: PlayerEnum,
    level: f32,
    rarity: f32,
    position: zhu.Vector2 = .zero,
    cost: f32 = 0,
};
```

初始化时从 `context.zon` 读取单位，计算实际 cost，再按 cost 排序。

```zig
pub var selected: ?usize = null;
pub var units: std.ArrayList(Unit) = .empty;
pub var unitLayoutDirty: bool = true;

pub fn init() void {
    for (contextZon.units) |zon| {
        var unit = zon;
        const base = spawn.playerZon[@intFromEnum(unit.class)].cost;
        const levelScale = 0.95 + 0.05 * unit.level;
        const rarityScale = 0.9 + 0.1 * unit.rarity;
        unit.cost = @round(base * levelScale * rarityScale);
        units.append(zhu.assets.allocator, unit) catch @panic("oom, can't append unit");
    }

    std.mem.sortUnstable(Unit, units.items, {}, struct {
        fn lessThan(_: void, a: Unit, b: Unit) bool {
            return a.cost < b.cost;
        }
    }.lessThan);
}
```

直接删除 `selected` 对应的单位，`orderedRemove` 会保持剩余单位顺序。

```zig
pub fn spendSelected() void {
    const index = selected.?;
    cost -= units.items[index].cost;
    _ = units.orderedRemove(index);
    selected = null;
    unitLayoutDirty = true;
}
```

## hud.zig

直接遍历 `ctx.units.items`。选择时保存的是单位索引，而不是职业枚举。

```zig
if (zhu.window.mouse.pressed(.LEFT)) {
    if (hoveredIndex) |idx| {
        if (ctx.cost >= ctx.units.items[idx].cost) {
            ctx.selected = idx;
            hoveredIndex = null;
        }
    }
}
```

增加 `arrangeUnits` 重新计算背景宽度和头像位置。

```zig
pub fn update() void {
    if (ctx.unitLayoutDirty) arrangeUnits();
    if (ctx.selected != null) return;

    // hover 和点击检测
}
```

准备态绘制也从选中索引取得单位，再读取职业类型。

```zig
if (ctx.selectedUnit()) |unit| drawPrepare(unit.class);
```

## spawn.zig

```zig
pub fn tryDeployPlayer(reg: *Registry, unit: ctx.Unit) void {
    const template = &playerZon[@intFromEnum(unit.class)];
    const mousePos = zhu.window.mousePosition;

    if (map.findPlace(template.attackKind, mousePos)) |idx| {
        if (ctx.cost < unit.cost) return;

        const place = &map.places.items[idx];
        const center = place.position.add(place.size.scale(0.5));

        const entity = doSpawn(reg, template);
        reg.add(entity, center);
        reg.add(entity, com.Player{});
        place.entity = entity;

        ctx.spendSelected();
        zhu.audio.playSound("assets/audio/Fantasy_UI (10).ogg");
        std.log.info("player deployed: {}", .{entity.index});
    }
}
```
