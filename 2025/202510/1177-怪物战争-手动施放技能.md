# 1177-怪物战争-手动施放技能

## 目标

让准备就绪的主动技能可以通过按钮或快捷键施放。技能施放后进入激活状态，并显示剩余持续时间。

## 环境

- Time 2026-04-28
- Zig 0.15.1

## 参考

1. <https://www.bilibili.com/video/BV1E8mCBBEmc/>

## 想法

可以手动实现释放技能，不过教程的这一节包含的内容真多。

## component.zig

技能增加 `Cast` 标签，用来表示本帧请求施放技能。

```zig
pub const skill = struct {
    pub const Ready = struct {}; // 技能准备完毕
    pub const Active = struct {}; // 技能激活中
    pub const Passive = struct {}; // 被动技能
    pub const Cast = struct {}; // 请求施放技能
};
```

## gui.zig

技能按钮只负责产生请求。按钮点击或按下 `S` 时，给实体添加 `com.skill.Cast`。

```zig
gui.igBeginDisabled(!ready);
const clicked = gui.igButton(value.name.ptr);
gui.igEndDisabled();
if (ready and (clicked or zhu.input.key.pressed(.S))) {
    reg.add(entity, com.skill.Cast{});
}
gui.igSameLine();
```

激活中显示剩余时间。这里暂时只读 `durationTimer`，真正推进持续时间放在技能系统中。

```zig
if (active) {
    if (passive) {
        _ = gui.igText("被动技能激活中");
    } else {
        const remaining = @max(0, value.duration - value.durationTimer);
        _ = gui.igText("激活中，剩余时间: %.1f 秒", remaining);
    }
}
```

## skill.zig

新增技能系统后，更新流程分成三个阶段：处理施放请求、推进冷却、推进持续时间。

```zig
pub fn update(reg: *zhu.ecs.Registry, delta: f32) void {
    updateCast(reg);
    updateCoolDown(reg, delta);
    updateDuration(reg, delta);
}
```

`updateCast` 遍历带有 `Skill` 和 `Cast` 的实体。

```zig
fn updateCast(reg: *zhu.ecs.Registry) void {
    var view = reg.view(.{ com.skill.Skill, com.skill.Cast });
    while (view.next()) |entity| {
        if (!view.has(entity, com.skill.Ready)) continue;
        if (view.has(entity, com.skill.Passive)) continue;

        const skill = view.getPtr(entity, com.skill.Skill);
        if (skill.passive) continue;

        skill.durationTimer = 0;
        view.remove(entity, com.skill.Ready);
        view.add(entity, com.skill.Active{});
    }

    reg.clear(com.skill.Cast);
}
```

持续时间推进也放在技能系统中。激活时间结束后，移除 `Active`，重置持续时间和冷却计时，技能重新进入冷却。

```zig
fn updateDuration(reg: *zhu.ecs.Registry, delta: f32) void {
    var view = reg.view(.{ com.skill.Skill, com.skill.Active });
    while (view.next()) |entity| {
        if (view.has(entity, com.skill.Passive)) continue;

        const skill = view.getPtr(entity, com.skill.Skill);
        if (skill.passive) continue;

        skill.durationTimer += delta;
        if (skill.durationTimer < skill.duration) continue;

        skill.durationTimer = 0;
        skill.coolDownTimer = 0;
        view.remove(entity, com.skill.Active);
    }
}
```

## 效果

技能冷却完成后，按钮变为可用。点击按钮或按 `S` 后，技能进入激活状态，面板显示剩余持续时间。
