# 1182-怪物战争-场景重启机制

## 目标

实现场景重启功能，支持重新开始当前关卡，以及通关/失败时显示提示。

## 环境

- Time 2026-05-02
- Zig 0.15.1

## 参考

1. <https://www.bilibili.com/video/BV171qQBPE7L/>

## 想法

可以重新启动场景。

## context.zig

`init` 不再重复填充 units 列表，改为调用 `reset`。

```zig
pub fn init() void {
    if (contextZon.level == 0) @panic("level must start at 1");
    levelIndex = @intCast(contextZon.level - 1);
    reset();
}
```

`reset` 重置所有全局状态并重建 units 列表，确保重启后可选单位恢复。

```zig
pub fn reset() void {
    cost = INITIAL_COST;
    homeHealth = INITIAL_HOME_HEALTH;
    // ...其他状态重置...

    units.clearRetainingCapacity();
    for (contextZon.units) |zon| {
        var unit = zon;
        const base = spawn.playerZon[@intFromEnum(unit.class)].cost;
        unit.cost = @round(spawn.statModify(base, unit.level, unit.rarity));
        units.append(zhu.assets.allocator, unit) catch @panic("oom");
    }
}
```

## scene.zig

新增 `restart`，依次重置状态、清空注册表、重新初始化生成和地图。

```zig
pub fn restart(reg: *zhu.ecs.Registry) void {
    ctx.reset();
    reg.reset();
    spawn.changeLevel(ctx.levelIndex);
    map.init(ctx.levelIndex);
}
```

## gui.zig

设置面板添加"重新开始"按钮。通关或失败时居中显示提示文字。

```zig
if (gui.igButton("重新开始")) {
    scene.restart(reg);
}
```

```zig
fn renderGameEnd() void {
    const label: ?[*:0]const u8 =
        if (ctx.isGameOver()) "失败"
        else if (ctx.isLevelClear()) "通关"
        else null;
    const text = label orelse return;
    // 居中显示...
}
```

## 效果

点击重新开始按钮后游戏重置到初始状态，通关或失败时屏幕显示对应文字。
