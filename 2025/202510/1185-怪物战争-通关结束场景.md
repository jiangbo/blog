# 1185-怪物战争-通关结束场景

## 目标

实现通关与结束场景，通关后延迟 3 秒进入结算，失败后进入结束场景，最后一关胜利直接进入胜利结束场景。

## 环境

- Time 2026-05-03
- Zig 0.15.1

## 参考

1. <https://www.bilibili.com/video/BV1GUBHBCEgt/>

## 想法

通关与结束场景。

## context.zig

`SceneState` 枚举添加 `clear` 和 `end`，新增 `win` 变量标记最终胜负，`reset` 中重置。

```zig
pub const SceneState = enum { title, battle, clear, end };
pub var win: bool = false;
```

## scene.zig

新增 `clearTimer` 和 `gameOverTriggered` 模块变量，`enter` 和 `restart` 中重置。通关奖励积分时启动 3 秒计时器，用 `isFinishedOnceUpdate` 倒计时结束后判断最后一关切换到 `end` 或 `clear`。

```zig
var clearTimer: ?zhu.Timer = null;
var gameOverTriggered: bool = false;
```

```zig
if (!ctx.levelClear and ctx.isLevelClear()) {
    ctx.levelClear = true;
    const reward = ctx.enemyKilledCount +
        @as(u32, @intCast(@max(0, ctx.homeHealth))) * 5;
    ctx.point += reward;
    clearTimer = .init(3);
}

if (clearTimer) |*t| {
    if (t.isFinishedOnceUpdate(delta)) {
        clearTimer = null;
        if (ctx.levelIndex + 1 >= map.maps.len) {
            ctx.win = true;
            ctx.pendingScene = .end;
        } else {
            ctx.pendingScene = .clear;
        }
    }
}

if (!gameOverTriggered and ctx.isGameOver()) {
    gameOverTriggered = true;
    ctx.win = false;
    ctx.pendingScene = .end;
}
```

## main.zig

`switchScene` 和 `frame` 改为 switch 语句，新场景只调用 `gui.update` + 黑色背景。

```zig
switch (ctx.currentScene) {
    .battle => {
        const scaled = delta * ctx.timeScale;
        gui.update(&registry, scaled);
        ctx.update(scaled);
        scene.update(&registry, scaled);
        hud.update(delta);
        zhu.batch.beginDraw(
            tiled.backgroundColor orelse .black);
        scene.draw(&registry);
        hud.draw();
    },
    .title => {
        gui.update(&registry, delta);
        title.update(delta);
        zhu.batch.beginDraw(
            tiled.backgroundColor orelse .black);
        title.draw();
    },
    .clear, .end => {
        gui.update(&registry, delta);
        zhu.batch.beginDraw(.black);
    },
}
```

## gui.zig

`update` 和 `draw` 按场景分支。新增 `renderLevelClear` 显示通关统计（关卡号、击杀数、基地血量、奖励积分、总积分）和"下一关"、"保存"、"返回标题"按钮。

```zig
fn renderLevelClear() void {
    gui.igSetNextWindowPos(
        .{ .x = 400, .y = 300 }, gui.ImGuiCond_Always);
    const flags = gui.ImGuiWindowFlags_NoTitleBar |
        gui.ImGuiWindowFlags_AlwaysAutoResize;
    if (gui.igBegin("Level Clear", null, flags)) {
        gui.igSetWindowFontScale(2.0);
        _ = gui.igText("通关");
        gui.igSetWindowFontScale(1.0);
        _ = gui.igText("关卡: %d", ctx.levelIndex + 1);
        _ = gui.igText("击杀: %d", ctx.enemyKilledCount);
        _ = gui.igText("基地血量: %d", ctx.homeHealth);
        const reward = ctx.enemyKilledCount +
            @as(u32, @intCast(@max(0, ctx.homeHealth))) * 5;
        _ = gui.igText("奖励积分: %d", reward);
        _ = gui.igText("总积分: %d", ctx.point);
        if (gui.igButton("下一关")) {
            ctx.levelIndex += 1;
            ctx.levelClear = false;
            ctx.pendingScene = .battle;
        }
        gui.igSameLine();
        if (gui.igButton("保存")) {
            showSavePanel = !showSavePanel;
        }
        gui.igSameLine();
        if (gui.igButton("返回标题")) {
            ctx.pendingScene = .title;
        }
    }
    gui.igEnd();
    if (showSavePanel) renderSavePanel();
}
```

新增 `renderEndScene` 根据 `ctx.win` 显示"胜利"或"失败"，提供"返回标题"和"退出"按钮。
