# 1171-怪物战争-出击部署

## 目标

实现出击区域占用验证、合法/非法位置视觉反馈、玩家死亡后释放出击点。

## 环境

- Time 2026-04-26
- Zig 0.15.1

## 想法

## map.zig

`Place` 增加 `entity` 字段记录占用实体，新增 `findPlace` 查找类型匹配且未占用的出击区域。

```zig
pub const Place = struct {
    position: zhu.Vector2,
    size: zhu.Vector2,
    kind: PlaceKind,
    entity: ?zhu.ecs.Entity = null,
};

/// 查找位置匹配且未被占用的出击区域
pub fn findPlace(kind: PlaceKind, pos: zhu.Vector2) ?usize {
    for (places.items, 0..) |place, i| {
        if (place.entity != null or place.kind != kind) continue;
        const rect: zhu.Rect = .init(place.position, place.size);
        if (rect.contains(pos)) return i;
    } else return null;
}
```

## spawn.zig

`tryDeployPlayer` 调用 `findPlace` 验证位置，合法则在区域中心生成单位并标记占用。`releasePlace` 供死亡系统调用，清空占用。

```zig
/// 尝试在合法出击区域部署玩家单位
pub fn tryDeployPlayer(reg: *Registry, playerEnum: com.PlayerEnum) void {
    const template = &playerZon[@intFromEnum(playerEnum)];
    const mousePos = zhu.window.mousePosition;

    if (map.findPlace(template.attackKind, mousePos)) |idx| {
        if (!ctx.canAfford(playerEnum)) return;

        const place = &map.places.items[idx];
        const center = place.position.add(place.size.scale(0.5));

        const entity = doSpawn(reg, template);
        reg.add(entity, center);
        reg.add(entity, com.Player{});
        place.entity = entity;

        ctx.spend(playerEnum);
    }
}

/// 释放被该实体占用的出击点
pub fn releasePlace(entity: zhu.ecs.Entity) void {
    for (map.places.items) |*place| {
        if (place.entity) |pe| {
            if (std.meta.eql(pe, entity)) {
                place.entity = null;
                return;
            }
        }
    }
}
```

## hud.zig

`drawPrepare` 根据 `findPlace` 结果，合法位置显示绿色，非法显示红色。

```zig
const found = map.findPlace(template.attackKind, mousePos);
zhu.batch.drawImage(sub, mousePos.add(template.offset), .{
    .color = if (found != null) .green else .red,
});
```

## death.zig

玩家死亡时调用 `releasePlace` 释放出击点。

```zig
if (!deadView.has(entity, com.Enemy)) {
    const e = deadView.toEntity(entity);
    spawn.releasePlace(e);
    reg.destroyEntity(e);
    continue;
}
```

## 效果

合法位置绿色、非法红色；占用区域不可再放置；玩家死亡释放出击点。
