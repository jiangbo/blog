## context.zig

新增积分和通关标记，reset 中重置。

```zig
pub var point: u32 = contextZon.point;
pub var levelClear: bool = false;
```

通关时在 `scene.zig` 中设置标记并奖励积分。

```zig
if (!ctx.levelClear and ctx.isLevelClear()) {
    ctx.levelClear = true;
    const reward = ctx.enemyKilledCount +
        @as(u32, @intCast(@max(0, ctx.homeHealth))) * 5;
    ctx.point += reward;
}
```

存档和读档使用 Zig 标准库的 `std.zon` 模块，序列化直接传
`units.items`，不需要中间类型转换。

```zig
pub fn saveGame(path: [:0]const u8) !void {
    const data = SaveData{
        .level = @intCast(levelIndex),
        .point = point,
        .levelClear = levelClear,
        .units = units.items,
    };

    var buf: [4096]u8 = undefined;
    var writer = std.Io.Writer.fixed(&buf);
    try std.zon.stringify.serialize(data, .{}, &writer);
    try zhu.window.saveAll(path, buf[0..writer.end]);
}

pub fn loadGame(path: [:0]const u8) !void {
    const content = try zhu.window.readAll(path);
    defer zhu.assets.allocator.free(content);

    const terminated = try std.fmt.allocPrintSentinel(
        zhu.assets.allocator, "{s}", .{content}, 0);
    defer zhu.assets.allocator.free(terminated);

    const data = try std.zon.parse.fromSlice(
        SaveData, zhu.assets.allocator, terminated, null, .{});
    defer std.zon.parse.free(zhu.assets.allocator, data);

    levelIndex = data.level;
    point = data.point;
    levelClear = data.levelClear;

    units.clearRetainingCapacity();
    for (data.units) |saveUnit| {
        units.append(zhu.assets.allocator, saveUnit) catch @panic("oom");
    }
}
```

ZON 是 Zig 的运行时序列化格式，语法就是 Zig 结构字面量。`readAll`
返回 `[]u8` 而 ZON 解析器需要 `[:0]const u8`，所以读档时用
`allocPrintSentinel` 加 null 终止符。

## gui.zig

标题场景的"确认角色"按钮切换角色信息表格，用 `BeginTable` 显示
所有角色的属性。属性通过 `spawn.statModify` 从模板和等级/稀有度
实时计算。每行末尾的升级按钮消耗积分。

```zig
for (ctx.units.items) |*unit| {
    const template = &spawn.playerZon[@intFromEnum(unit.class)];
    const hp = spawn.statModify(
        template.stats.maxHealth, unit.level, unit.rarity);
    // ...
    if (canUpgrade and clicked) {
        ctx.point -= upgradeCost;
        unit.level += 1;
        unit.cost = @round(spawn.statModify(
            template.cost, unit.level, unit.rarity));
    }
}
```

标题场景"载入游戏"按钮和战斗场景设置面板的"保存"按钮分别切换
读档和存档面板，各有 3 个槽位。

## 效果

标题场景可查看和升级角色，战斗通关后获得积分奖励。存档保存为 ZON
文件，读档可恢复进度。
