# 1174-怪物战争-显示单位信息

## 目标

接入 ImGui 调试面板，鼠标悬停单位时显示属性提示，左键选中玩家单位后显示固定信息窗口和攻击范围。

## 环境

- Time 2026-04-27
- Zig 0.15.1

## 参考

1. <https://www.bilibili.com/video/BV1Dvm5BrEj6/>

## 想法

将之前 hud 中显示范围的单独提取成了一个系统，便于选择显示范围复用。

## component.zig

新增单位显示信息相关组件。

```zig
pub const ShowRange = struct {}; // 显示攻击范围标签
pub const Name = struct { value: [:0]const u8 }; // 名称组件
pub const ClassName = struct { value: [:0]const u8 }; // 职业名称组件
```

`Stats` 中增加等级和稀有度。这里使用 `f32`，和关卡、角色配置中的数据类型保持一致。

```zig
pub const Stats = struct {
    health: i32,
    maxHealth: i32,
    attack: i32,
    defense: i32,
    level: f32 = 1,
    rarity: f32 = 1,
};
```

## context.zig

关卡上下文中保存当前悬停实体和选中实体。

```zig
pub var hoveredEntity: ?zhu.ecs.Entity = null;
pub var selectedEntity: ?zhu.ecs.Entity = null;
pub var uiWantCaptureMouse: bool = false;
```

角色槽位增加 `name` 字段，用来生成玩家实体的 `Name` 组件。

```zig
pub const Unit = struct {
    name: [:0]const u8,
    face: u32,
    class: PlayerEnum,
    level: f32,
    rarity: f32,
    position: zhu.Vector2 = .zero,
    cost: f32 = 0,
};
```

## context.zon

角色姓名参考默认会话数据填写。职业名称仍然来自 `player.zon` 中的模板名称。

```zig
.{
    .name = "加尔隆",
    .face = 0,
    .class = .warrior,
    .level = 1,
    .rarity = 1,
},
```

## spawn.zig

模板名称作为 `ClassName` 添加。敌人没有个体姓名，只显示敌人类型；玩家实体在出击时额外添加 `Name`。

```zig
reg.add(entity, zon.stats);
reg.add(entity, com.ClassName{ .value = zon.name });
```

敌人的等级和稀有度来自关卡配置。

```zig
var stats = reg.get(entity, com.Stats);
stats.level = levels[ctx.levelIndex].enemyLevel;
stats.rarity = levels[ctx.levelIndex].enemyRarity;
reg.getPtr(entity, com.Stats).* = stats;
```

玩家部署时使用角色槽位中的姓名、等级和稀有度。

```zig
var stats = reg.get(entity, com.Stats);
stats.level = unit.level;
stats.rarity = unit.rarity;
reg.getPtr(entity, com.Stats).* = stats;

reg.add(entity, com.Name{ .value = unit.name });
```

## selection.zig

新增选择系统，集中处理悬停、选中、范围标签维护，以及选择调试绘制。

```zig
pub fn update(reg: *zhu.ecs.Registry, _: f32) void {
    if (ctx.uiWantCaptureMouse) {
        ctx.hoveredEntity = null;
    } else {
        updateHoveredEntity(reg);
        updateSelectedEntity(reg);
    }

    updateShowRange(reg);
}
```

悬停检测使用统一的 64x64 判定框，并在重叠时选择 y 值更大的实体。

```zig
const center = pos.addY(-32);
const rect: zhu.Rect = .init(center.sub(.xy(32, 32)), .xy(64, 64));

if (rect.contains(mousePos) and pos.y > maxY) {
    maxY = pos.y;
    ctx.hoveredEntity = view.toEntity(index);
}
```

选中只允许玩家单位。敌人可以悬停查看信息，但左键不会进入选中状态。

```zig
if (zhu.window.mouse.pressed(.LEFT)) {
    const entity = ctx.hoveredEntity orelse return;
    if (!reg.has(entity, com.Player)) return;
    ctx.selectedEntity = entity;
} else if (zhu.window.mouse.pressed(.RIGHT)) {
    ctx.selectedEntity = null;
}
```

`ShowRange` 标签由选择系统维护，绘制时只遍历带标签的实体。

```zig
var view = reg.view(.{ com.ShowRange, com.Position, com.attack.Range });
while (view.next()) |entity| {
    const pos = view.get(entity, com.Position);
    const range = view.get(entity, com.attack.Range);
    const circle = zhu.getImage("circle.png");
    const r = range.v;
    zhu.batch.drawImage(circle, pos.sub(.xy(r, r)), .{
        .size = .xy(r * 2, r * 2),
        .color = .rgba(1, 1, 1, 0.3),
    });
}
```

为了方便观察判定范围，调试绘制也放在选择系统中。白色表示普通实体，紫色表示悬停，黄色表示选中。

## gui.zig

ImGui 每帧创建新帧后，绘制基础窗口、悬停提示和选中窗口，再记录 ImGui 是否希望捕获鼠标。

```zig
renderHoveredUnit(reg);
renderSelectedUnit(reg);

const io = gui.igGetIO();
ctx.uiWantCaptureMouse = io.*.WantCaptureMouse;
```

悬停窗口从实体组件读取信息。玩家有姓名，敌人没有姓名，只显示类型名称。

```zig
if (reg.tryGet(entity, com.Name)) |name| {
    _ = gui.igText("%s  ", name.value.ptr);
    gui.igSameLine();
}

if (reg.tryGet(entity, com.ClassName)) |className| {
    _ = gui.igText("%s", className.value.ptr);
}
```

选中窗口固定在左上角，显示更完整的信息。近战单位额外显示阻挡数量。

```zig
gui.igSetNextWindowPos(.{ .x = 10, .y = 10 }, gui.ImGuiCond_Always);
const flags = gui.ImGuiWindowFlags_NoTitleBar |
    gui.ImGuiWindowFlags_AlwaysAutoResize;
if (gui.igBegin("角色状态", null, flags)) {
    // 显示属性
}
```

## scene.zig

`scene` 不再直接处理实体悬停和选中，只调用选择系统。部署准备单位仍然留在场景中处理。

```zig
system.selection.update(reg, delta);
if (!ctx.uiWantCaptureMouse) {
    if (zhu.window.mouse.pressed(.LEFT)) {
        if (ctx.selectedUnit()) |unit| {
            spawn.tryDeployPlayer(reg, unit);
        }
    } else if (zhu.window.mouse.pressed(.RIGHT)) {
        if (ctx.selected != null) ctx.selected = null;
    }
}
```

绘制时在血条和投射物之后调用选择系统，显示攻击范围和调试判定框。

```zig
system.health.draw(reg);
system.projectile.draw(reg);
system.selection.draw(reg);
```

## main.zig

`Registry` 上移到 `main.zig`，然后传给 `gui` 和 `scene`。

```zig
var registry: zhu.ecs.Registry = undefined;

pub fn frame(delta: f32) void {
    gui.update(&registry, delta);
    ctx.update(delta);
    scene.update(&registry, delta);
}
```

## 效果

鼠标悬停单位时会显示 tooltip，左键选中玩家单位后会显示左上角状态窗口，并绘制攻击范围。
