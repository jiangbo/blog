# 1176-怪物战争-技能冷却计时

## 目标

让主动技能部署后进入冷却，冷却结束后添加 `Ready` 标签，并在 ImGui 中显示冷却进度和就绪状态。

## 环境

- Time 2026-04-28
- Zig 0.15.1

## 参考

1. <https://www.bilibili.com/video/BV1E8mCBBEmc/>

## 想法

AI 真强大。

## component.zig

现有计时器事件只处理攻击冷却，这里给 `Timer` 增加技能冷却和技能持续时间两种类型。

```zig
pub const Timer = struct { // 计时器
    remaining: f32,
    entity: zhu.ecs.Entity,
    type: enum { attack, skillCoolDown, skillDuration },
};
```

本次只实现 `skillCoolDown`，`skillDuration` 先作为后续施放技能时的入口。

## spawn.zig

部署玩家单位时不再直接添加技能组件，而是通过 `addSkill` 统一处理技能初始状态。

```zig
reg.add(entity, com.Name{ .value = unit.name });
reg.add(entity, center);
reg.add(entity, com.Player{});
if (template.skill) |skill| addSkill(reg, entity, skill);
```

被动技能只添加 `Passive` 标签。

```zig
fn addSkill(reg: *Registry, entity: zhu.ecs.Entity, skill: com.skill.Skill) void {
    reg.add(entity, skill);

    if (skill.passive) {
        reg.add(entity, com.skill.Passive{});
        return;
    }

    if (skill.coolDown <= 0) {
        reg.add(entity, com.skill.Ready{});
        return;
    }

    reg.addEvent(com.Timer{
        .remaining = skill.coolDown,
        .entity = entity,
        .type = .skillCoolDown,
    });
}
```

## timer.zig

计时器事件到期后，根据类型添加对应标签。

```zig
switch (timer.type) {
    .attack => reg.add(timer.entity, com.attack.Ready{}),
    .skillCoolDown => {
        if (reg.tryGetPtr(timer.entity, com.skill.Skill)) |skill| {
            skill.coolDownTimer = skill.coolDown;
        }
        reg.add(timer.entity, com.skill.Ready{});
    },
    .skillDuration => {},
}
```

为了让 ImGui 进度条能看到变化，每帧还会推进主动技能的 `coolDownTimer`。

```zig
fn updateSkillCoolDown(reg: *zhu.ecs.Registry, delta: f32) void {
    var view = reg.view(.{com.skill.Skill});
    while (view.next()) |entity| {
        if (view.has(entity, com.skill.Ready)) continue;
        if (view.has(entity, com.skill.Passive)) continue;

        const skill = view.getPtr(entity, com.skill.Skill);
        if (skill.passive) continue;

        skill.coolDownTimer = @min(skill.coolDown, skill.coolDownTimer + delta);
    }
}
```

## gui.zig

```zig
const progress = if (value.coolDown > 0)
    value.coolDownTimer / value.coolDown
else
    0;
gui.igProgressBar(progress, .{ .x = 120, .y = 0 }, null);
```
