# 1132-阳光岛-受伤时闪烁

## 目标

实现角色受伤时闪烁效果。

## 环境

- Time 2026-01-30
- Zig 0.15.1

## 参考

1. <https://www.bilibili.com/video/BV1jf9XYQEhW>

## 想法

之前的太困难了，现在做一个简单的。

## player.zig

查看一下计时器的值就行了。

```zig
pub fn draw() void {
    if (hurtTimer.isRunning()) { // 受伤时闪烁效果
        if (hurtTimer.isEvenStep(0.15)) state.draw();
    } else state.draw();
}
```
