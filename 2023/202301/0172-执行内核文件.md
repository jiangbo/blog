# 0172-执行内核文件

## 环境

- Time 2022-11-11
- WSL-Ubuntu 22.04
- QEMU 6.2.0
- NASM 2.15.05

## 前言

### 说明

参考：<https://os.phil-opp.com/multiboot-kernel/>

### 目标

将编写好的内核可执行文件，制作成 ISO 镜像，然后从 QEMU 启动。

## 建立文件目录

```text
isofiles
└── boot
    ├── grub
    │   └── grub.cfg
    └── kernel.bin
```

## grub.cfg

```cfg
set timeout=0
set default=0

menuentry "my os" {
    multiboot2 /boot/kernel.bin
    boot
}
```

## 制作 ISO

安装依赖：`apt-get install grub-pc-bin grub-common xorriso`
